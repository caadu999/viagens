Projeto desenvolvido para praticar conceitos básicos de roteamento no Next.js.

--------- Funcionalidades -----------

- Página inicial com apresentação da agência.
- Página de destinos com lista de destinos turísticos.
- Navegação entre páginas utilizando rotas do Next.js.
- Componentização básica com React.
- Dados mockados para simular informações de viagens.

--------- Tecnologias -----------

- Next.js
- React
- TypeScript
- CSS Modules

--------- Como Executar -----------

- Clone o repositório
- Acesse a pasta do projeto
- Instale as dependências
- Execute o servidor de desenvolvimento
- Ele rodará no http://localhost:3000
